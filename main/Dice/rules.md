# ROLLING the DICE ... (Simulation)

This is a simple `Dice-Rolling` game based on LUCK!

Roll infinitely, till you are exhausted!



### MODUS OPERANDI

* Type -- `Y` or `YES`, to continue rolling ..!
* Type -- `N` or `NO`, to exit.
